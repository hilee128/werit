차트가 뭔지부터

출생 차트는 내가 태어난 순간 하늘의 행성들이 어디에 있었는지를 기록한 지도야. 장소, 날짜, 시간이 1분만 달라져도 차트가 바뀜.

태양자리만 아는 건 차트의 10%도 안 봐. 달, 금성, 상승점(ASC)을 같이 봐야 진짜 보이기 시작해.

태양자리는 내가 되고 싶은 것.
달자리는 내가 이미 느끼는 것.
Step 01

차트 뽑는 법

01
astro-seek.com 접속
astro-seek.com → Birth Chart 클릭. 무료야.
02
생년월일 + 시간 + 장소 입력
출생 시간이 제일 중요해. 모르면 부모님한테 물어보거나 주민등록 발급 서류에 나오는 경우 있어.
03
Positions 탭 + Aspects 탭 복사
차트 이미지 캡처 + 텍스트 데이터 둘 다 저장해. 분석할 때 텍스트가 핵심이야.
04
Claude에 붙여넣기
이미지 + 텍스트 같이 올리고 분석 요청하면 됨.
시간을 모를 때
정오(12:00)로 설정하면 달자리 정도는 확인 가능해. 단, ASC(상승점)는 틀릴 수 있어.
Step 02

Step 03

Step 04

애스펙트가 뭘 말하는지

행성들이 서로 어떤 각도에 있느냐. 협력하는 관계인지, 충돌하는 관계인지.

트라인 △ (120도)
자연스럽게 흘러. 노력 없이도 되는 것들. 재능에 가까움.
섹스타일 ✶ (60도)
기회. 조금만 노력하면 연결되는 것들.
스퀘어 □ (90도)
긴장, 충돌. 힘들지만 극복하면 가장 강해지는 부분. 숙제.
오포지션 ☍ (180도)
두 에너지가 반대 방향에서 당김. 균형 잡는 게 숙제.
컨정션 ☌ (0도)
두 행성이 합쳐짐. 엄청 강해지거나 충돌하거나 둘 중 하나.
오브 (Orb) 숫자
각도 옆에 Orb 숫자가 있어. 숫자가 작을수록 영향이 강함. 3도 이하면 진짜 타이트하게 작동하는 거야.
Step 05

Claude한테
분석 요청하는 법

차트 캡처 + Positions/Aspects 텍스트 복사해서 같이 올리고 이 프롬프트 쓰면 돼.

첨부한 차트 데이터 기반으로 분석해줘.

1. 달이랑 금성 사인/하우스/애스펙트 교차 분석
2. 달 → 감정적으로 마음 열리는 실제 조건
3. 금성 → 무의식적으로 끌리는 것들. 이상형 말고 진짜로
4. 달이랑 금성 충돌하는지 강화하는지

꾸미지 말고 날카롭게. 말투는 친한 친구한테 설명하듯이.
